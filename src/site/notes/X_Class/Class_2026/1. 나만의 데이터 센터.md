# 나만의 데이터 도서관 만들기
## AI 시대, 홍보인들을 위한 데이터 수집, 정제, 활용 고도화

## 서론: AI가 거짓말쟁이가 되는 이유

친구에게 "어제 뉴스에서 삼성 주가 얼마나 올랐다던데?"라고 물어봤는데, 친구가 "응, 10% 올랐어!"라고 답했다고 해보자. 그런데 나중에 확인해보니 실제로는 3%밖에 안 올랐다면? 친구가 거짓말한 게 아니라 정확히 기억하지 못했을 뿐이다.

AI도 마찬가지다. ChatGPT에게 "우리 회사 브랜드 이미지 어때?"라고 물어보면 그럴듯한 답을 해준다. 하지만 막상 확인해보면 존재하지도 않는 신문 기사를 인용하거나, 3년 전 데이터를 최신 정보라고 말하기도 한다.

30년 PR 일을 하면서 깨달은 것: 잘못된 정보로 만든 보고서는 독이다. 특히 우리가 다루는 브랜드 이미지나 기업 평판은 한 번 잘못되면 돌이키기 어렵다.

#AI거짓말 #AI환각 

## 1. AI 거짓말 방지하기: 정확한 재료로 요리해야 맛있는 음식이 나온다

### 1-1. AI는 왜 거짓말을 할까?

음식점 비유로 생각해보자:

*   재료를 확인 안 하는 요리사: 냉장고에 뭐가 있는지 제대로 안 보고 요리한다
    결과: 상한 재료로 요리해서 배탈 나는 음식 완성

AI도 마찬가지다:

*   존재하지 않는 뉴스: "○○신문에서 보도했다"며 가짜 기사 인용
*   엉터리 여론조사: 실제 조사 안 하고 추측으로 "국민 70%가 좋아한다" 발표
*   경쟁사 가짜 정보: 근거 없이 "경쟁사 매출이 20% 감소했다" 주장

### 1-2. 정보 수집이 왜 중요한가?

집 짓기 과정으로 비교해보자:

🏗️ 잘못된 방식: AI야, 집 지어줘!
└── 결과: 기초 공사 없이 지은 집 → 무너짐

🏗️ 올바른 방식:

1.  땅 고르기 (정보 수집) / 크롤링
2.  기초 공사 (검증) / csv 
3.  벽돌 쌓기 (분석) / plan
4.  완성된 집 (보고서) / result

각 단계에서 우리가 직접 확인할 수 있다:

*   수집: 어느 사이트에서, 언제, 어떤 단어로 찾았는지 기록 / #cursor #Antigravity 
*   검증: 뉴스 진짜인지, 날짜 맞는지 직접 확인 / *csv파일로 저장 원본 링크 확보*
*   분석: 검증된 자료만 가지고 AI에게 분석 요청 / *csv 파일만으로 검증 분석 요청*
*   보고서: 모든 자료 출처가 명확한 믿을 만한 결과 / *저장 CSV파일로만 결과물 생성

![class.jpg](/img/user/Present_PR/Clients/_assets/class.jpg) 

**수집 (크롤링) --> 검증 (CSV) --> 분석 (Plan) --> 보고서 (Report)**

### 1-3. 실제로 어떻게 다를까?

❌ 기존 방식 (위험한 방법): "삼성전자 브랜드 이미지 분석해줘" → AI가 인터넷 어디서든 자료 찾아와서 분석 → 결과: 근거 없는 애매한 이야기

✅ 개선된 방식 (안전한 방법):

1.  최근 3개월 주요 신문사 삼성 관련 기사만 수집
2.  네이버, 다음 뉴스 댓글 반응 수집
3.  삼성 공식 SNS 게시물과 댓글 수집
4.  이렇게 모은 확실한 자료로만 AI 분석 → 결과: 어디서 가져온 정보인지 명확한 믿을 만한 분석

## 2. 나만의 정보 금고 만들기: 돈 모으듯 정보도 모으자

### 2-1. 정보 저장고가 왜 필요할까?

은행 적금 비유:

*   매달 10만원씩 적금 → 1년 후 120만원 + 이자
*   매일 좋은 정보 수집 → 1년 후 경쟁사가 못 따라올 정보 자산

우리 PR 대행사가 잘나가는 이유는? 

**남들이 모르는 정보를 우리만 알고 있기 때문**이다.

==체계적으로 정보를 모으면==

*   시간 여행 가능: 작년 이맘때 브랜드 이미지 어땠는지 비교
*   패턴 발견: "아, 이런 이슈 터지면 항상 이렇게 반응하네!"
*   경쟁사 비교: 우리 vs 경쟁사 정확한 순위 측정
*   미래 예측: 과거 데이터로 "다음엔 이런 일 일어날 것 같은데?" 예상

#과거데이터발굴 #패턴발견 #현상발견 #경쟁사비교 #트레드예측 

### 2-2. AI 거짓말 완전 차단하는 방법

**도서관 사서 비유:**

*   일반 도서관: 모든 책 다 빌려줌 (진짜 책도, #가짜책도)
*   나만의 도서관: 우리가 검증한 책만 보관, AI는 여기서만 찾기 #우리만의도서관 #우리회사만의도서관

**폐쇄형 정보 시스템 구축:**

*   AI가 검색할 수 있는 범위를 우리 금고 안으로만 제한
*   인터넷에서 마음대로 검색하지 못하게 차단
*   모든 답변에 "이 정보는 우리 금고의 ○○ 폴더에서 가져왔습니다" 표시

### 2-3. RAG = 나만의 똑똑한 비서

**RAG를 쉽게 설명하면: 검색해서 답하는 시스템 = 옵시디언**

AI 비서 비교:

| 구분     | LLM 일반 비서  | 나만의 AI (RAG)  |
| ------ | ---------- | ------------- |
| 정보 출처  | 인터넷 아무데서나  | 우리 회사 자료실에서만  |
| 검색 방식  | 단어만 맞으면 OK | 브랜드 연관성까지 고려  |
| 답변 스타일 | 일반적인 대답    | 오랜 학습을 통한 답변  |
| 신뢰성 확인 | 출처만 표시     | 내부 자료 기반의 신뢰도 |

## 3. Obsidian으로 나만의 데이터 도서관 만들기

### 3-1. 왜 Obsidian일까?

**집 (Obsidian) vs 전세방 (LLM) 비유:**

*   클라우드 서비스 = 전세방: 언제 나가라고 할지 모름, 집주인 마음대로
*   Obsidian = 내 집: 완전히 내 것, 언제든 접근 가능, 아무도 못 건드림. 
  #로컬폴더 #로컬데이터 #나만의RAG #개인RAG #개인데이터센터 #마크다운 

장점:

*   완전한 소유권: 클라이언트 #기밀정보도 안전하게 보관
*   관계 지도 그리기: 브랜드-이슈-경쟁사-언론사 연결고리 한눈에 보기
*   과거와 현재 연결: "작년에 비슷한 사건 있었는데, 그때 어떻게 했더라?" 
  
  #완전한소유권 #데이터관계망 #과거현재연결 #관계지도그리기 #10년전키워드 #10년후키워드 

### 3-2. PR 전용 정리함 구조

아파트 구조처럼 체계적으로 정리:

```
🏢 우리 회사 정보 주상 복합 아파트
├── 🏠 1층: 고객사별 방
│   ├── 🚪 삼성방
│   │   ├── 📺 언론보도 TV
│   │   ├── 🔍 경쟁사 분석 돋보기
│   │   └── 🎯 캠페인 결과 게시판
│   └── 🚪 LG방
├── 🏠 2층: 업계 정보 연구소
│   ├── 📈 트렌드 분석실
│   ├── 💭 소비자 감정 측정실
│   └── 📋 정부 규제 정보실
├── 🏠 3층: 언론 관계 관리소
│   ├── 👨‍💼 기자 프로필 수첩
│   ├── 📰 언론사 분석 파일
│   └── 📅 보도 일정표
├── 🏠 4층: 위기관리 상황실
│   ├── 📚 과거 사례 도서관

    ├── 🏆 대박난 캠페인들
    ├── 📖 실패에서 배운 교훈들
    └── 💰 투자 대비 효과 계산기
```

### 3-3. 정보 수집부터 활용까지 3단계

요리하는 과정으로 비교:

1.  **1단계: 장보기 (정보 수집)**

    *   네이버 뉴스 API = 정육점 (브랜드 관련 기사 자동 수집)
    *   포털 댓글 스크래핑 = 시장 (사람들 반응 엿듣기)
    *   경쟁사 사이트 크롤링 = 다른 집 엿보기 (경쟁사 동향 파악)
    
2.  **2단계: 냉장고 정리 (Obsidian 저장)**

    ```markdown
    # 삼성 갤럭시S24 관련 뉴스_2024년 3월 15일

    ## 어디서 가져왔나?
    - **출처**: 조선일보 경제면
    - **언제**: 2024.03.15
    - **태그**: #삼성 #갤럭시S24 #신제품 #소비자반응

    ## 핵심 내용
    갤럭시S24 첫 주 판매량이 작년보다 25% 늘었다...

    ## 연관 정보
    [[작년 갤럭시S23 성과]] - [[아이폰15와 비교]] - [[올해 스마트폰 시장 전망]]
    ```
3.  **3단계: 요리하기 (AI 분석)**

    *   우리 냉장고(Obsidian) 재료만 사용해서 AI가 요리(분석)
    *   고객별 맞춤 요리(보고서) 자동 완성
    *   과거 레시피(유사 사례)와 비교해서 더 맛있게

### 3-4. 실제 상황에서 써보기

🚨 상황 1: 우리 회사에 안 좋은 뉴스 터졌다!

1.  Obsidian에서 "비슷한 사건" 즉시 검색
2.  그때 어떻게 대응했는지, 결과는 어땠는지 확인
3.  AI가 현재 상황에 맞는 대응책 제안
4.  모든 제안의 근거 = 우리가 쌓아온 경험 데이터

💼 상황 2: 새 고객사 미팅 준비하기

1.  해당 업계 트렌드 자료 Obsidian에서 뽑기
2.  경쟁사들 성공/실패 사례 정리
3.  우리만 아는 특별한 정보로 깜짝 놀랄 제안서 만들기

## 4. AI 시대:  PR 전문가는 어떻게 일할까?

### 4-1. AI 활용 데이터 수집, 분석 활용 전문가 = PR 전문가

부동산 VS PR:

*   10년 전: 강남 땅 미리 사둔 사람 = 부자
*   10년 후: 양질의 데이터(정보) 미리 모아둔 PR 전문가 = 생존자

단계별 경쟁력:

*   #대행사 차원: 모든 고객 데이터 합쳐서 업계 전체 흐름 파악
*   회사 #홍보팀: 우리 브랜드만의 완벽한 #정보_저장고
*   #개인: 내 전문 분야 깊이 있는 정보로 인정받는 전문가


## 결론: 데이터 활용 마스터 = PR 전문가

AI 시대에 살아남는 PR 전문가의 비밀은? 
AI와 싸우지 말고 AI가 제대로 일할 수 있는 환경을 만들어주는 것이다.

3단계 데이터 분석 마스터:

1.  체계적 정보 수집: #크롤링, #스크래핑, #API로 보물 찾기
2.  #Obsidian 지식 창고: 내 것만의 완벽한 정보 저장소
3.  #RAG 기반 분석: 검증된 정보로만 AI와 협업

이 모든 것이 하나로 연결될 때, 우리는 AI 시대에도 없어서는 안 될 PR 전문가가 된다.

정보의 바다에서 진주를 찾아내는 능력. 이것이 바로 차세대 PR 전문가의 핵심 무기가 될 것이다.
