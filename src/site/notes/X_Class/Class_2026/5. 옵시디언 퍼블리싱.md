## 🌳 개념 이해: 내 방의 노트를 세상에 알리는 법 (비유)

우리가 방금 한 작업은 총 3가지 도구를 연결한 것입니다.

1. **옵시디언 (내 책상 위의 비밀 일기장):** 나만 볼 수 있는 일기를 쓰는 곳입니다.
    
2. **깃허브 (글을 보관하는 안전한 금고):** 내 일기장 내용을 복사해서 넣어두는 튼튼한 금고입니다. 전 세계 사람들과 공유할 준비를 하는 곳이죠.
    
3. **넷리파이 (금고 속 글을 전시하는 전시장):** 금고에 있는 글을 예쁘게 꾸며서 사람들이 볼 수 있게 길가에 전시하는 전시장(웹사이트)입니다.
    

---

## 🛠️ 단계별 실행 매뉴얼 (하나씩 따라하기)

### **1단계: 깃허브(GitHub)에 내 금고 만들기**

- **할 일:** 남이 만든 템플릿(설계도)을 내 계정으로 가져오기.
    
- 방법: 1. 깃허브 사이트에서 원본 창고를 찾습니다.
    
    2. 오른쪽 상단의 'Fork' 버튼을 누릅니다. (이것은 남의 요리법을 내 요리책으로 복사해오는 것과 같습니다.)
    
    3. **'Create fork'**를 누르면 내 이름표(admonth1224)가 붙은 내 금고가 생깁니다.
    

### **2단계: 넷리파이(Netlify) 전시장 오픈하기**

- **할 일:** 내 금고와 전시장을 연결하기.
    
- 방법: 1. Netlify 홈페이지에 가서 GitHub 계정으로 로그인합니다.
    
    2. [Add new site] → **[Import an existing project]**를 누릅니다.
    
    3. GitHub를 선택하고, 내가 Fork 해온 금고 이름(obsidian-digital-garden)을 클릭합니다.
    
    4. 아래의 **[Deploy site]**를 누릅니다. 이제 전시장 주소가 생깁니다! (예: pai-git.netlify.app)
    

### **3단계: 열쇠(Token) 복사하기**

- **할 일:** 옵시디언이 내 금고(깃허브)에 글을 넣을 수 있게 허락하는 열쇠 만들기.
    
- 방법: 1. GitHub 우측 상단 프로필 클릭 → Settings로 갑니다.
    
    2. 맨 아래 Developer settings를 누릅니다.
    
    3. Personal access tokens (classic) → Generate new token을 누릅니다.
    
    4. Note에 이름을 쓰고, repo 박스에 꼭 체크합니다. (금고를 열 수 있는 권한입니다.)
    
    5. 생성된 ghp_...로 시작하는 긴 코드를 메모장에 복사합니다. (한 번만 보여주니 꼭 저장하세요!)
    

### **4단계: 옵시디언과 금고 연결하기**

- **할 일:** 옵시디언 앱에 금고 주소와 열쇠 입력하기.
    
- 방법: 1. 옵시디언 설정 → Digital Garden 플러그인에 들어갑니다.
    
    2. GitHub Username에 내 아이디, token에 복사한 열쇠를 넣습니다.
    
    3. Base URL에 넷리파이 전시장 주소를 넣습니다.
    
    4. 상단에 **"Connected with full access"**라는 초록 불이 들어오면 성공입니다!
    

### **5단계: 글 전시하기 (퍼블리싱)**

- **할 일:** 특정 일기(노트)를 세상에 내놓기.
    
- 방법: 1. 노트의 **'속성(Properties)'**을 엽니다.
    
    2. dg-publish 항목을 만들고 **체크(V)**를 합니다.
    
    3. 옵시디언 왼쪽의 정원(Digital Garden) 아이콘을 누릅니다.
    
    4. [Publish] 버튼을 클릭하면 금고를 거쳐 전시장에 내 글이 나타납니다!

## ❓ 자주 묻는 질문 (빠짐없이 답변)

- **무료인가요?** 네, 위 방식은 깃허브와 넷리파이의 무료 플랜을 사용하므로 돈이 전혀 들지 않습니다.
    
- **왜 안 올라가나요?** `dg-publish: true`가 문서의 **가장 맨 윗부분**에 있지 않으면 옵시디언이 "아, 이건 비밀 일기구나"라고 생각해서 전시하지 않습니다.
    
- **제목을 바꾸고 싶어요:** 파일 이름을 바꾸고 다시 **[Publish]**를 누르면 전시장에도 새 제목으로 바뀝니다.
